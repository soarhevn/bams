輔助說明: 會員資料編輯 - Edit Member

Details about the member are edited on this page. The ID number cannot be edited as this number is the reference for all associated records for this members.

Dates are required to be entered in the format, 'year-month-day' or the system will give you an error.

Once created, members records cannot be deleted, as there is other data for the member that is linked to the member record, If a member is not longer active, then mark the member as inactive on this page.
