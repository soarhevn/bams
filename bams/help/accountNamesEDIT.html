輔助說明: 編輯更改帳戶名稱 - Edit Account Name

Any changes made to the account will reflect across any transactions using this account!

編碼

共有四碼。
  • 第一碼
    • A.為代表經常會費。
    • B.代表勞保費。
    • C.代表健保費。
    • D.代表互助金。
  • 第二碼
    • 01.代表資產: 包括銀行帳戶,應收帳款,手頭現金。
    • 02.代表負債:包括應付帳款。
    • 03.代表收入.
    • 05.代表支出.
  • 第三碼--為照手冊上所編制之"項",或隨機增加編碼。
  • 第四碼--為照手冊上所編制之"目",或隨機增加編碼。

科目類別

Type of account must be choosen.

帳目組

Assign account to a group for ease of bookkeeping and accounting.

預設之交叉抵消帳戶

This is the default account that most transactions will settle against. Whenever a transaction is recorded, this default cross account will show up for the account, but another can be selected. For example, if the ADSL account is usually paid directly from the 經常會費 Bank account, then select the 經常會費 Bank as the default cross account; if an ADSL transaction does not happen to be paid from the 經常會費 Bank account, but from cash on hand, cash on hand can be selected when recording that transaction.

帳目終止

Because accounts are assigned to many transactions, they cannot be deleted. If an account will no longer be used, mark it as 帳目終止 and will no longer show up in the selection list for new transactions. Old transactions using the deprecated account will still show correctly.
